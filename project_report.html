Project Overview

SecondBrain is a comprehensive web-based reminder management system designed to help users stay organized and productive. The application provides an intuitive interface for creating, managing, and tracking reminders with advanced features like voice notes, email notifications, and admin controls.

The system addresses the common problem of forgetting important tasks and appointments by providing a centralized platform where users can create reminders with flexible scheduling options, voice recordings, and automated notifications.

Project Objectives

  • Develop a user-friendly reminder management system with modern UI/UX
  • Implement voice recording functionality for personalized reminders
  • Create automated email notification system using cron jobs
  • Build comprehensive admin panel for user and system management
  • Ensure secure user authentication and data protection
  • Implement responsive design for mobile and desktop compatibility

Key Features

User Management

Complete user registration, login, profile management with secure authentication

Database Design

The application uses MySQL database with the following main tables:

users
  • id (Primary Key, Auto Increment)
  • name (VARCHAR 255)
  • email (VARCHAR 255, Unique)
  • password (VARCHAR 255, Hashed)
  • profile_picture (VARCHAR 255)
  • is_admin (TINYINT, Default 0)
  • created_at (TIMESTAMP)
reminders
  • id (Primary Key, Auto Increment)
  • user_email (VARCHAR 255, Foreign Key)
  • title (VARCHAR 255)
  • description (TEXT)
  • reminder_date (DATE)
  • reminder_time (TIME)
  • voice_file (VARCHAR 255)
  • sent_status (TINYINT, Default 0)
  • created_at (TIMESTAMP)
  • updated_at (TIMESTAMP)
complaints
  • id (Primary Key, Auto Increment)
  • user_email (VARCHAR 255)
  • subject (VARCHAR 255)
  • message (TEXT)
  • status (ENUM: pending, resolved, in_progress)
  • created_at (TIMESTAMP)
  • updated_at (TIMESTAMP)

Implementation Details

Frontend Implementation

  • Modern responsive design using CSS Grid and Flexbox
  • Font Awesome icons for enhanced UI elements

Notification System

Challenges & Solutions

Voice Recording Browser Compatibility

Challenge: Ensuring consistent voice recording functionality across different browsers.

Solution: Implemented fallback mechanisms and used MediaRecorder API with proper error handling.

Email Delivery Reliability

Challenge: Ensuring reliable email delivery with attachments.

Solution: Configured Gmail SMTP with proper authentication and implemented comprehensive error logging.

File Upload Security

Challenge: Securing voice file uploads from malicious content.

Solution: Implemented file type validation, size limits, and secure file storage practices.

Cron Job Scheduling

Challenge: Setting up automated reminder checking on server.

Solution: Created PHP script for cron execution with detailed logging and error handling.
